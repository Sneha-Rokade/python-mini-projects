# 📒 Contact Book Application (Python CLI)

## 🚀 Project Overview

The Contact Book Application is a command-line based Python project designed to manage personal contacts efficiently. It allows users to add, view, search, and delete contacts with proper validation and persistent storage.

This project demonstrates real-world software engineering practices including modular architecture, data handling, validation, and logging.

---

## 🛠️ Tech Stack

* Python
* JSON (for data storage)
* File Handling
* Logging module

---

## ✨ Features

* Add new contact (Name, Phone, Email)
* View all saved contacts
* Search contact by name
* Delete contact
* Input validation (Email & Phone)
* Persistent storage using JSON
* Logging for tracking operations

```

3. Run the application:

```
