# Password Validator (Python)

## Project Overview

This is a beginner-friendly **Password Validation System** built using Python as part of my learning journey to strengthen problem-solving and backend logic.

The goal of this project is to simulate how real-world systems validate user passwords and enforce security rules.

---

## Features Implemented (V1)

* Password length validation
* Uppercase & lowercase letter check
* Password strength classification:
  * Weak
  * Good
  * Strong
* Simple and readable logic using loops and conditions
